# MERN E-Commerce Platform

A full-stack E-Commerce web application built with the **MERN Stack (MongoDB, Express.js, React, and Node.js)**. It provides a seamless shopping experience with category-wise product browsing, shopping cart functionality, user authentication, and an admin panel for product management.

## Features

- User Authentication
- Category-wise Products (Men, Women, Kids)
- Featured Products & New Arrivals
- Product Details Page
- Shopping Cart
- Admin Dashboard
- Responsive Design
- MongoDB Integration
- RESTful APIs

## Tech Stack

**Frontend:** React.js, Vite, CSS, Bootstrap

**Backend:** Node.js, Express.js

**Database:** MongoDB, Mongoose

## Installation

```bash
git clone https://github.com/SrishtiAnand/mern-ecommerce-platform.git

cd frontend
npm install
npm run dev

cd ../backend
npm install
npm start
```
